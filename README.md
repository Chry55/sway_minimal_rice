# swayfx_minimal_rice
A minimal Wayland rice focused for productivity

![Screenshot](Wall.png)
![Screenshot](Tiling.png)
![Screenshot](Rofi.png)

* Shell: Zsh
* WM: swayfx
* Launcher: wofi
* Terminal: kitty
* File manager: thunar
* Status bar: waybar
* Text editor: neovim
* Terminal: kitty
* Terminal theme: dracula (credit: https://draculatheme.com/kitty)
* Text editor: neovim with custom config (install vim-plug first)
